# HDPE Pipes Product Experience Website

This project is a modern industrial product landing page built to showcase HDPE pipes and fittings. The focus was on creating a clean interface, structured content flow, and interactive elements suitable for real-world B2B use.

## Live Demo

https://syedsiddiq104.github.io/industrial-pipes-website/

## Repository

https://github.com/syedsiddiq104/industrial-pipes-website

## Features

* Structured product layout with clear information hierarchy
* Image carousel with thumbnail navigation
* Modal-based forms for quote requests and catalogue downloads
* Responsive design across mobile, tablet, and desktop
* Reusable UI components such as cards, tabs, and badges
* Simple and efficient JavaScript for interactivity

## Tech Stack

* HTML5
* CSS3
* JavaScript

## Approach

The project was designed with a focus on usability and clarity. The layout follows a typical product-focused flow, starting from overview, features, technical specifications, and ending with strong call-to-action sections. Interactive elements like modals and carousels were implemented to improve user engagement without overcomplicating the experience.
